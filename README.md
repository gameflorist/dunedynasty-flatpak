# Flatpak configuration for Dune Dynasty

This repository houses the flatpak configuration and build-process for [Dune Dynasty](https://github.com/gameflorist/dunedynasty), a remaster / enhancement of the classic real-time strategy game Dune II by Westwood Studios.

The releases can be downloaded from the main project's [release page](https://github.com/gameflorist/dunedynasty/releases).
